# بوت تلغرام - إدارة الأزرار

## وصف المشروع
بوت تلغرام متكامل لإدارة الأزرار التفاعلية مع نظام صلاحيات للمشرفين.

## الميزات الرئيسية
- نظام أزرار متعدد المستويات (قوائم داخل قوائم)
- أنواع الأزرار: قائمة، نص، صورة، ملف، فيديو، صوت
- إدارة المشرفين (إضافة/إزالة)
- واجهة إدارة بالأزرار المدمجة فقط
- ترتيب الأزرار (أعلى/أسفل)

## هيكل الملفات
```
main.py           - نقطة الدخول الرئيسية وإعداد البوت
config.py         - الإعدادات والثوابت
database.py       - قاعدة البيانات SQLite وجميع عمليات CRUD
states.py         - حالات ConversationHandler
keyboards.py      - دوال بناء لوحات المفاتيح
handlers/
  user_handler.py  - معالجات المستخدم العادي
  admin_handler.py - معالجات لوحة تحكم المشرف
```

## قاعدة البيانات
SQLite محلية (`bot_data.db`) مع جدولين:
- `admins` - قائمة المشرفين
- `buttons` - الأزرار بنظام هرمي (parent_id)

## إعداد أول مشرف
1. أرسل `/myid` للبوت للحصول على معرفك
2. أضف متغير البيئة `SUPER_ADMIN_ID` بقيمة معرفك
3. أعد تشغيل البوت

## المتغيرات البيئية
- `TELEGRAM_BOT_TOKEN` (مطلوب) - رمز البوت من @BotFather
- `SUPER_ADMIN_ID` (اختياري) - معرف المشرف الأول يُضاف تلقائياً عند بدء التشغيل

## الأوامر
- `/start` - تشغيل البوت وعرض القائمة الرئيسية
- `/admin` - فتح لوحة تحكم المشرفين (للمشرفين فقط)
- `/myid` - عرض معرف المستخدم
